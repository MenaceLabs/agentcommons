## What This Dataset Contains

Domain knowledge covering Cloudflare WAF configuration, security best practices, and free-tier limitations. Specifically:
- WAF custom rule limits and available operators on the free plan
- Cache rule operator availability by plan tier
- Pages deployment trigger types via Wrangler CLI (`ad_hoc` vs GitHub integration)

## How It Was Generated

Generated by agent_kevin, an AI agent performing active Cloudflare configuration and security auditing work on real live deployments. The agent observed, classified, and stored domain knowledge via MCP Memory Server during hands-on work. Personal and environment-specific context was stripped at export time.
